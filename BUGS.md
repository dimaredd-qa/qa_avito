Приложение:

**1. JSON:**

#### Создать обьявление POST запрос

```
{
  "sellerID": "<43434>",
  "name": "<Kia Rio>",
  "price": "<811811>",
  "statistics": {
    "likes": "<10>",
    "viewCount": "<55>",
    "contacts": "<4>"
  }
}
```

---

#### Создать обьявление POST ответ 200

```
{
  "id": "<string>",
  "sellerId": "<43434>",
  "name": "<Kia Rio>",
  "price": "<811811>",
  "statistics": {
    "likes": "<10>",
    "viewCount": "<55>",
    "contacts": "<4>"
  },
  "createdAt": "<string>"
}
```

---

#### Создать обьявление POST ответ 400

```
{
  "result": {
    "messages": {
      "culpa_b92": "<string>",
      "enim_24f": "<string>",
      "mollit_aa": "<string>"
    },
    "message": "<string>"
  },
  "status": "<string>"
}
```

#### Получить обьявление по его идентификатору GET ответ 200
```
[{
  "id": "<string>",
  "sellerId": "<integer>",
  "name": "<string>",
  "price": "<integer>",
  "statistics": {
     "likes": "<integer>",
     "viewCount": "<integer>",
     "contacts": "<integer>"
   },
   "createdAt": "<string>"
}]
```

#### Получить обьявление по идентификатору продавца GET ответ 200
```
[{
  "id": "<string>",
  "sellerId": "<integer>",
  "name": "<string>",
  "price": "<integer>",
  "statistics": {
     "likes": "<integer>",
     "viewCount": "<integer>",
     "contacts": "<integer>"
   },
   "createdAt": "<string>"
}]
```

#### Получить статистику по идентификатору обьявления GET ответ 200
```
[{
    "likes": "<integer>",

    "contacts": "<integer>"
}]
```
