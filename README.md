Вполне допускаю, что в некоторых тест-кейсах моё решение может показаться избыточным, а в других, возможно, какие-то аспекты могли быть упущены. # Задание 2.1.Проект автотестов для API сервиса объявлений

Автоматизированные тесты для проверки REST API сервиса размещения и управления объявлениями.

## Технологии

- **Python 3.8+**
- **Pytest** - фреймворк для тестирования
- **Requests** - для HTTP-запросов
- **Allure** - для генерации отчетов о тестировании
- **Faker** - для генерации тестовых данных

## Установка и настройка

### 1. Клонирование репозитория
```
git clone https://github.com/dimaredd-qa/qa_avito.git
```
### 2. Установка зависимостей
```
pip install -r requirements.txt
```
### 3. Запуск всех тестов c подробным результатом
```
pytest -vv
```
### 4. Запуск тестов с генерацией Allure отчетов
```
pytest --alluredir=allure-results
```
### 5. Просмотр Allure отчета
```
allure serve allure-results
```
### 5. Запуск конкретных тестовых файлов
```
pytest test_create_ad.py -v
pytest test_get_ad_for_id.py -v
pytest test_get_by_seller_id.py -v
pytest test_get_statistics_by_id_ad.py -v
```
